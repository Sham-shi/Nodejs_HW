## Практическое задание 3

**Тема:** Знакомство с Vite

## Задача

Создать небольшой одностраничный проект на Vite

## Выполнение

### 1. Создать проект через Vite

#### 1. Открыть терминал в среде разработки.
#### 2. Создать проект через Vite последней версии:

```bash
npm create vite@latest
```

#### 3. Ввести в терминале имя проекта.
#### 4. Выбрать фреймворк `Vanila`.
#### 5. Выбрать вариант `JavaScript`.
#### 6. В ответ на вопрос `Install with npm and start now?` выбрать `Yes`.
#### 7. Убедиться, что стартовая страница открывается в браузере.
#### 8. Закрыть локальный сервер нажатием сочетания клавиш `Ctrl` + `C`.

### 2. Очистить шаблон

Удалить лишние демонстрационные элементы, которые создаются по умолчанию.

Нужно оставить:

- базовую HTML-структуру;
- подключение основного JS-файла;
- файл стилей.

### 3. Инициализировать npm-проект

Для автоматического создания файла `package.json` со значениями по умолчанию выполнить команду:

```bash
npm init -y
```

Для создания файла `package.json` с ручным заполнением полей:

```bash
npm init
```

### 3. Создать файлы проекта

- с помощью интерфейса создать основной файл проекта `app.js`;
- создать папку `utils`;
- создать файл `message.js`, в котором написать функцию возвращающую текст приветствия, например:

```json
function greeting() {
    let greetText = 'Приветствую!';
    return greetText;
}
```

- экспортировать эту функцию в модуль:

```bash
module.exports = {greeting};
```

### 4. Подключить собственный модуль в основной файл

Импортировать функцию `greeting` в `app.js` из `message.js`:

```json
const greet = require("./utils/message");
```

### 5. Установить внешний пакет `chalk`

`Chalk` — это популярная Node.js библиотека для цветного форматирования текста в терминале.

Установить:

```bash
npm install chalk
```

Для проверки установленного пакета:

```bash
npm list
```

Подключить модуль в `app.js`:

```json
const chalk = require('chalk').default;
```

### 6. Заполнить основной файл проекта

В файле `app.js` подключить встроенные модули Node.js:

- `path`;
- `os`.

С помощью подключенных модулей (в том числе `chalk`, `./utils/message`) вывести в консоль:

- приветствие из `message.js`;
- версию Node.js;
- текущие дату и время в своем регионе;
- путь к текущей папке;
- путь к текущему фалу;
- имя операционной системы.

### 7. Добавить npm-скрипт

В `package.json` добавить раздел:

```json
"scripts": {
  "start": "node app.js"
}
```

### 8. Запуск проекта

Для запуска проекта можно использовать:

```bash
node app.js
```

или

```bash
npm start
```
