## Практическое задание 3  

**Тема:** Знакомство с Vite

## Задача

Создать небольшой одностраничный проект на Vite, представляющий собой меню ресторана с функцией добавления блюд в корзину.

## Выполнение

### 1. Создать проект через Vite

- Открыть терминал в среде разработки.

- Создать проект через Vite последней версии:

```bash

npm  create  vite@latest

```

- Ввести в терминале имя проекта.

- Выбрать фреймворк `Vanila`.

- Выбрать вариант `JavaScript`.

- В ответ на вопрос `Install with npm and start now?` выбрать `Yes`.

- Убедиться, что стартовая страница открывается в браузере.

- Закрыть локальный сервер нажатием сочетания клавиш `Ctrl` + `C`.

### 2. Очистить шаблон

Удалить лишние демонстрационные элементы, которые создаются по умолчанию.

Нужно оставить:

- базовую HTML-структуру;
- подключение основного JS-файла;
- файл стилей.

### 3. Реализовать функционал приложения

В проекте реализован следующий функционал:

-   **Отображение меню:** Список блюд (Борщ, Карбонара, Цезарь и др.) с названием, описанием, ценой и фотографией загружается из файла `data.js`.
    
-   **Добавление в корзину:** При клике на кнопку "В корзину" под каждым блюдом, информация о нем добавляется в массив корзины, а общая стоимость пересчитывается.
    
-   **Отображение корзины:** Содержимое корзины динамически отображается в виде таблицы, общая сумма обновляется в реальном времени.
    
-   **Очистка корзины:** Кнопка "Очистить" удаляет все добавленные блюда и обнуляет общую стоимость.

### 4. Структура проекта

  После выполнения проекта структура файлов выглядит следующим образом:

```
practice3
├── node_modules
├── public
│   ├── favicon.svg
│   └── icons.svg
└── src
 ├── assets          # Папка с изображениями блюд
 │   ├── borsch.jpg
 │   ├── carbonara.jpg
 │   ├── cesar.jpg
 │   ├── cheesecake.png
 │   └── margherita.jpg
 ├── data.js         # Данные меню и логика работы с корзиной
 ├── main.js         # Главный файл приложения, рендеринг HTML и обработчики
 ├── render.js       # Функции для отрисовки элементов (блюд и строк корзины)
 ├── style.css       # Стили для приложения
 ├── index.html
 ├── package-lock.json
 ├── package.json
 └── README.md
 ```

### 5. Запуск проекта

1.  Запустить сервер разработки:
    
    ``` bash
    
    npm run dev
    
    ```
    
2.  Открыть ссылку, которая появится в терминале (обычно `http://localhost:5173`) сочетанием клавиши `Ctrl` и нажатием левой кнопкой мыши, чтобы увидеть результат.
    

## Результат

Создано одностраничное приложение (`SPA`) на чистом JavaScript с использованием Vite. Приложение позволяет ознакомиться с ассортиментом ресторана, собрать заказ и увидеть его итоговую стоимость. Проект демонстрирует работу с модулями (`import/export`), динамическим созданием DOM-элементов и обработкой событий.
